本项目是是基于DeepSeek实现的用于展示游戏王爱好者DIY卡牌的静态页面。

Json格式说明
  id：卡片序号（数字）
  name：卡名（字符串）
  name_extra：额外卡名（字符串）
    | 额外卡名指的是卡名中无法展现的其他卡名或字段，仅用于搜索，不会显示在卡片详情页中。
  series：系列（字符串）
    | 系列并不是指字段，而是这个卡牌属于什么原作或者属于哪个分类。
  type：卡片种类（怪兽、魔法、陷阱）（字符串）
  attribute：属性（字符串）
  race：种族（字符串）
  monster_type：怪兽种类（字符串）（可拥有复数种类）
  spell_trap_type：魔陷种类（字符串）
  level：等级/阶级（根据是否为超量怪兽显示为不同文本）（数字）
  scale：灵摆刻度（数字）
  link：连接值（数字）
  atk：攻击力（数字）
  def：守备力（数字）
  desc：效果文本（也用于通常怪兽的描述性文字）（字符串）
  pendulum_desc：灵摆效果（字符串）
  image：图像地址（字符串）
  images：复数图像地址（字符串）
  incomplete: 是否为未完成卡片（布尔值）
      | 若为true，则该卡片在搜索界面有白色遮罩，详细界面无法点开卡片大图。
